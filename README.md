# ORTHOVIA 🎮📚

## Overview
**Orthovia** is an educational game application designed to enhance learners’ understanding of **homophones and homographs** through interactive and competitive gameplay. The system integrates game-based learning strategies to improve spelling accuracy, contextual understanding, and student engagement.

This project was developed as part of an academic thesis.

---

## 🎯 Objectives
- To improve learners’ spelling competence in homophones and homographs
- To promote active and engaging learning through gamification
- To provide immediate feedback for better retention and understanding
- To support teachers with an interactive instructional tool

---

## 🎮 Game Modes

### 🔹 Single Player Mode
- The learner competes against a computer-generated opponent (AI)
- A preliminary question determines who selects a tile first
- The player selects from a **25-number tile wall**
- Each tile contains a hidden question with varying difficulty levels
- Correct answers move the player forward on a **10-step progression board**
- The first to reach the final step wins

### 🔹 Double Player Mode
- Two learners compete against each other
- Follows the same mechanics as Single Mode
- Encourages collaboration and competitive learning

---

## 🧩 Game Mechanics
- 25-number interactive tile system
- Difficulty levels: Easy, Average, Difficult (revealed after selection)
- Multiple-choice question format
- Real-time feedback (correct/incorrect responses)
- Progress tracking through a step-based advancement system

---

## 🛠️ Technologies Used
- Replit (Development Environment)
- HTML, CSS, JavaScript (Web Application)
- (Optional) Unity Game Engine (for future development)
- GitHub (Version Control and Repository Management)

---

## 🚀 Deployment
The application is deployed as:
- A web-based application accessible via browser
- An Android application using WebView technology (APK format)

Due to academic constraints, the application is not published on the Google Play Store but is distributed via direct installation and web access.
